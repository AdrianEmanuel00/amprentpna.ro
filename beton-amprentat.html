Ce primesti

O suprafata continua, decorativa si gandita pentru utilizare reala.

Betonul amprentat este potrivit cand vrei aspect de piatra, ardezie, dale sau caramida, dar cu o placa turnata continuu. Rezultatul bun depinde de pregatirea terenului, calitatea turnarii, alegerea pigmentilor si sigilarea finala.

Amprent PNA calibreaza lucrarea dupa destinatie: curte cu masina, alee pietonala, terasa, zona de piscina sau acces tehnic.

Modele si culori

Piatra naturala, ardezie, dale, caramida, antracit, gri, crem, maro sau tonuri oxid, alese in functie de fatada si peisaj.

Strat suport

Compactarea, pantele si grosimea influenteaza durabilitatea mai mult decat orice promisiune generica de pret.

Protectie finala

Sigilarea protejeaza culoarea, reduce absorbtia si ofera un finisaj mai usor de intretinut.

Etape de executie

  1. 01Evaluare

    Localitate, suprafata, poze, acces si destinatia spatiului.

  2. 02Pregatire

    Strat suport, cofraje, pante, armare unde este necesar.

  3. 03Turnare si amprentare

    Beton, pigment, release powder, textura si rosturi controlate.

  4. 04Sigilare

    Protectie finala si recomandari de intretinere dupa predare.

Vrei o oferta corecta?

Trimite localitatea, suprafata aproximativa, tipul lucrarii si 3 poze cu terenul.

Trimite detaliile
Cat dureaza o lucrare de beton amprentat?

Depinde de suprafata, pregatire, acces si vreme. Pentru incadrare rapida, trimite suprafata si poze.

Este potrivit pentru curte cu trafic auto?

Da, daca stratul suport, grosimea, armarea, pantele si rosturile sunt proiectate pentru incarcare auto.

Cum se intretine?

Se curata periodic si se resigileaza la nevoie, mai ales in zone cu trafic intens sau expunere la saruri.
